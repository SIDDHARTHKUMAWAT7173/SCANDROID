# Mobile App Security Analyzer

A static analysis tool to detect common Android security vulnerabilities in APK files.

## Features

- Manifest misconfiguration detection
- Dangerous permission analysis
- Hardcoded secret detection
- Weak cryptography detection
- Insecure TLS detection
- OWASP Mobile Top 10 mapping
- Risk scoring
- Structured JSON report output
